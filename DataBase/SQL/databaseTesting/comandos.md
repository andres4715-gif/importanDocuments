# _Comandos_

- Para usar la db:

```shell
$ use <Nombre de la base de datos>
$ use classicModels;
```

- Para consultar toda la data de una tabla especifica:

```shell
$ select * from <Nombre de la tabla>
$ select * from employees;
```

- Para consultar las tablas que tiene la base de datos:

```shell
$ show tables;

- Results:
`customers`,
`employees`,
`offices`,
`orderdetails`,
`orders`,
`payments`,
`productlines`,
`products`
```
